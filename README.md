NLQ SQL Chatbot — Natural Language to SQL with AI

An AI-powered chatbot that converts **natural language questions** into **SQL queries**, executes them on a **PostgreSQL** database, and visualizes results with **interactive dashboards**.

> 🔗 Built on top of the [Ecommerce ETL Pipeline](https://github.com/Deepinalama/ecommerce-etl-pipeline) — demonstrating an end-to-end workflow from data engineering to AI-powered analytics.

---

 Features

-  **Natural Language → SQL** — Ask questions in plain English, get SQL queries generated by an LLM
-  **PostgreSQL Integration** — Executes AI-generated queries directly on your database
-  **Interactive Dashboard** — Visualizes query results with charts and tables
-  **Groq LLM** — Fast inference using Groq's API for query generation
- **Schema-aware** — Model understands your database schema for accurate query generation

 How It Works

```
User types question in plain English
        ↓
Schema sent to Groq LLM as context
        ↓
LLM generates SQL query
        ↓
Query executed on PostgreSQL
        ↓
Results displayed as table + chart
```

 Getting Started

### Prerequisites
- Python 3.9+
- PostgreSQL database
- Groq API key (free at [console.groq.com](https://console.groq.com))

 1. Clone the repo
```bash
git clone https://github.com/Deepinalama/nlq-sql-chatbot.git
cd nlq-sql-chatbot
```

 2. Install dependencies
```bash
pip install -r requirements.txt
```

 3. Set up environment variables
```bash
cp .env.example .env
```

Edit `.env`:
```
GROQ_API_KEY=your_groq_api_key
DB_HOST=localhost
DB_PORT=5432
DB_NAME=your_database
DB_USER=your_user
DB_PASSWORD=your_password
```

 4. Run the app
```bash
streamlit run app.py
```

App will be available at `http://localhost:8501`

 Example Queries

| Natural Language | Generated SQL |
|---|---|
| "Show top 10 customers by revenue" | `SELECT customer_id, SUM(amount) FROM orders GROUP BY customer_id ORDER BY SUM(amount) DESC LIMIT 10` |
| "How many orders were placed last month?" | `SELECT COUNT(*) FROM orders WHERE order_date >= NOW() - INTERVAL '1 month'` |
| "Which products have low stock?" | `SELECT name, stock FROM products WHERE stock < 10` |

# NLQ SQL Chatbot

An AI-powered chatbot that converts natural language into SQL queries and executes them on a PostgreSQL database.

# Features

- Natural language to SQL
- PostgreSQL integration
- AI-powered query generation
- Dashboard with charts

# Tech Stack

- Python
- PostgreSQL
- Groq LLM


# Project Relationship

This project is built on top of my Ecommerce Data Pipeline project.

The previous project focused on:
- Extracting and transforming nested JSON data
- Designing and populating a PostgreSQL database
- Building an analytics dashboard

This repository focuses on:
- Natural Language → SQL conversion using an LLM
- Executing AI-generated SQL queries
- Interactive data exploration and visualization

Together, these projects demonstrate an end-to-end workflow from data engineering to AI-powered analytics.
